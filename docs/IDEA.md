# AIEventRadar 아이디어

## 한 줄 설명
국내외 AI 관련 행사 정보를 자동으로 수집해 사용자에게 알림을 보내주는 서비스.

## 문제 (Problem)
- AI 행사·밋업·해커톤 정보가 **Festa, 이벤터스, Luma, Dev-Event, Devpost** 등 여러 플랫폼에 흩어져 있다
- 각 플랫폼을 매번 직접 방문해 검색해야 함 → 놓치는 경우 다반사
- 같은 행사가 여러 플랫폼에 중복 게시되어 정보 위생도 낮음
- "AI" 키워드 외에 LLM/생성형/딥러닝 등 관심 분야별 필터링이 어려움

## 해결 (Solution)
1. **다중 소스 자동 수집**: Festa, 이벤터스, Luma, Dev-Event(GitHub), Devpost
2. **정규화 + 중복 제거**: 제목+날짜+장소 기반 dedup
3. **AI 키워드 필터링**: LLM/ML/생성형/딥러닝/RAG/Agent 등
4. **개인화 알림**: 사용자 관심 키워드 기반 새 행사 / 마감임박 푸시

## 타겟 사용자
- AI/ML 엔지니어, 데이터 사이언티스트
- AI 분야 학생/연구자
- AI 스타트업 종사자, PM, 디자이너
- 해커톤·컨퍼런스 적극 참여자

## 데이터 소스 (확정)
| 소스 | 유형 | 수집 방식 |
|------|------|-----------|
| Festa (festa.io) | 국내 개발자 밋업 | 스크래핑 |
| 이벤터스 (event-us.kr) | 국내 B2B 컨퍼런스/전시 | 스크래핑 |
| Luma (luma.com) | 글로벌 AI 커뮤니티 | 비공식 API or 스크래핑 |
| Dev-Event (GitHub) | 큐레이션된 한국 개발자 행사 | README 파싱 |
| Devpost | 해커톤 | 공식 페이지 파싱 |

## 핵심 기능 (MVP 후보)
1. 행사 수집기 (소스별 어댑터 + 통합 스케줄러)
2. 정규화 스키마 + 중복 제거 로직
3. AI 키워드 필터링
4. 행사 리스트 웹 UI (검색/필터/북마크)
5. 알림 채널 (이메일/Slack/Discord/Telegram 중 1~2개로 시작)

## 비전 / Phase 2
- 다국어 지원
- 추천 알고리즘 (사용자 행동 기반)
- 캘린더 연동 (Google Calendar, iCal)
- 후기/평점 시스템
- 행사 호스트 직접 등록 기능
